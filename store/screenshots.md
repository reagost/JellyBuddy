## General Guidelines

- **iOS**: 6.7" (iPhone 15 Pro Max): 1290 x 2796 px, 6.5" (iPhone 14 Plus): 1284 x 2778 px

- **Format**: PNG or JPEG, no alpha channel
- **Quantity**: 6-8 screenshots per platform
- **Style**: Device frame + colored background + headline text overlay
- **Background colors**: Alternate between primary purple (#6C63FF), dark (#1A1A2E), teal (#00BFA6), and warm (#FF6B6B)

## Text Overlay Style

- **Headline font**: Bold, 48-56pt, white color with slight shadow
- **Description font**: Regular, 24-28pt, white color with 80% opacity
- **Positioning**: Headline centered above the device frame, description below the headline
- **Device frame**: Use official Apple/Android device frames (iPhone 15 Pro / Pixel 8)
- **Drop shadow**: Subtle shadow on device frame (offset: 0, 4px; blur: 20px; color: rgba(0,0,0,0.15))

---

## Production Notes

- Capture all screenshots at the highest resolution available
- Use the same demo data across all screenshots for consistency (e.g., same username, XP level, streak count)
- For AI chat screenshot, show a realistic and helpful conversation
- Ensure dark mode screenshots have sufficient contrast
- Test text readability on both light and dark backgrounds
- For the Chinese version, use Simplified Chinese characters
- Provide both Chinese and English versions of each screenshot for respective store listings
